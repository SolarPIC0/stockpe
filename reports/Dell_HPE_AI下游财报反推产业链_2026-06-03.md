# Dell 与 HPE 财报后大涨：从 AI 产业链下游反推上游景气

研究日期：2026-06-03  
财报口径：Dell Technologies FY2027 Q1；Hewlett Packard Enterprise FY2026 Q2。  
说明：本文不是投资建议。财报和指引优先采用公司官网/IR 官方披露；股价涨幅使用行情与新闻报道交叉验证。

## 1. 结论

Dell 和 HPE 财报后的暴涨，本质是市场确认了三件事：

1. AI 服务器需求不是停留在芯片端，而是已经转化为整机、存储、网络、服务和企业部署收入。
2. AI 基础设施需求仍然超过供应，瓶颈从 GPU 扩展到内存、CPU、硬盘、网络和周边零部件。
3. 公司上调全年指引，使市场必须重算 FY2026/FY2027 EPS，而不是只把它当成单季超预期。

用我们刚建立的“季报触发系统”看，Dell/HPE 都属于强触发：不只是业绩好，而是全年盈利模型被改写。

## 2. Dell：为什么股价狂飙

Dell FY2027 Q1 直接打穿了市场原本的盈利模型。

关键数据：

| 指标 | FY2027 Q1 | 同比 |
|---|---:|---:|
| 总收入 | 438.42 亿美元 | +88% |
| GAAP EPS | 5.24 美元 | +282% |
| 非 GAAP EPS | 4.86 美元 | +214% |
| 经营现金流 | 40.81 亿美元 | +46% |
| ISG 收入 | 290.09 亿美元 | +181% |
| AI 优化服务器收入 | 161.32 亿美元 | +757% |
| AI 订单 | 244 亿美元 | - |
| AI backlog | 513 亿美元 | 创纪录 |

最关键的不是单季收入，而是指引上修：

| 指引项目 | FY2026 Q4 发布时 | FY2027 Q1 后 | 变化 |
|---|---:|---:|---:|
| FY2027 总收入中值 | 1400 亿美元 | 1670 亿美元 | +270 亿美元 |
| FY2027 AI 优化服务器收入 | 约 500 亿美元 | 约 600 亿美元 | +100 亿美元 |
| FY2027 非 GAAP EPS 中值 | 12.90 美元 | 17.90 美元 | +38.8% |

这就是股价反应的核心：市场不是只奖励 Q1，而是把全年利润预期从 12.90 美元 EPS 级别重算到 17.90 美元级别。

Dell 的电话会还给出几个重要下游信号：

- AI 客户数超过 5000，且过去 6 个月增长超过 50%。
- Pipeline 是 backlog 的数倍，且跨 neocloud、主权 AI、企业客户。
- 需求超过供应，主要约束是 memory。
- 除 memory 外，公司也提到 NAND、DRAM、CPU、硬盘以及一篮子周边部件压力。
- 传统服务器收入同比 +92%，说明不是只有 GPU 训练集群，企业传统算力刷新和 AI inference 也在拉动需求。

这说明 Dell 已经从“卖硬件的低估值公司”被市场重新看成“AI 工厂整合商”。

## 3. HPE：为什么也暴涨

HPE 的逻辑和 Dell 不完全一样。Dell 更像 AI 服务器订单爆发；HPE 更像“AI + Networking + Juniper 整合 + 成本协同”同时重估。

关键数据：

| 指标 | FY2026 Q2 | 同比/变化 |
|---|---:|---:|
| 总收入 | 106.78 亿美元 | +40% |
| GAAP 毛利率 | 36.5% | +810 bps |
| 非 GAAP 毛利率 | 36.9% | +750 bps |
| GAAP EPS | 0.44 美元 | 高于原指引 0.09-0.13 |
| 非 GAAP EPS | 0.79 美元 | 高于原指引 0.51-0.55 |
| 经营现金流 | 14.10 亿美元 | 同比增加 19 亿美元 |
| 自由现金流 | 9.15 亿美元 | 同比增加 18 亿美元 |
| Networking 收入 | 26.90 亿美元 | +148.2% |
| Data Center Networking | 3.20 亿美元 | +233.3% |
| Cloud & AI 收入 | 77.07 亿美元 | +22.9% |
| Cloud & AI 服务器收入 | 55 亿美元 | +32.7% |

更重要的是指引：

| 项目 | Q1 后指引 | Q2 后指引 | 变化 |
|---|---:|---:|---:|
| FY2026 收入增长 | 17%-22% | 29%-33% | 大幅上调 |
| FY2026 GAAP EPS | 1.02-1.22 美元 | 2.42-2.52 美元 | 大幅上调 |
| FY2026 非 GAAP EPS | 2.30-2.50 美元 | 3.35-3.45 美元 | 大幅上调 |
| FY2026 自由现金流 | 至少 20 亿美元 | 至少 35 亿美元 | 提前达到原 FY2028 目标 |
| FY2027 框架 | - | 收入增长 8%-12%，非 GAAP EPS 增长 12%-16%，FCF 至少 45 亿美元 | 给出延续性 |

HPE 的股价暴涨来自一个更强的叙事切换：市场原来把它看成传统企业 IT/服务器公司，现在开始把它看成 AI 基础设施、数据中心网络和企业私有 AI 的受益者。

## 4. 从 Dell/HPE 下游反推 AI 产业链

Dell/HPE 位于 AI 产业链最下游：它们把 GPU、CPU、HBM/DRAM、NAND、硬盘、光模块、交换机、PCB、机柜、电源、液冷、软件和服务打包成客户能部署的系统。

所以它们的财报是很好的下游验证器。

### 4.1 对存储和内存：偏强

Dell 明确说需求超过供应，primary constraint 是 memory；电话会还提到 DRAM、NAND、CPU、硬盘等都在承压。

推论：

- 美光、三星、SK 海力士的高端 DRAM/HBM/NAND 景气仍有支撑。
- 服务器整机厂库存和备货会放大上游波动，所以要看订单是否是真实部署，而不是渠道囤货。
- 如果 Dell/HPE 后续继续上调 AI server 指引，存储周期的盈利高点可能继续被延后。

### 4.2 对 PCB/覆铜板：偏强但要看供给扩张

AI 服务器、B200/B300/GB200/GB300、交换机、加速卡、背板、电源管理、存储系统都会提升 PCB 层数、材料等级和高速信号要求。

推论：

- 沪电股份、生益科技这类高端 PCB/覆铜板公司，与 Dell/HPE 的下游爆发存在逻辑联动。
- Dell 提到客户越来越关注基础设施密度和数据中心空间效率，这通常会推高高端板材、背板、高速连接、热管理和电源设计要求。
- 但沪电调研纪要也提示同行扩产、同质化竞争和利润空间挤压风险，不能只看需求。

### 4.3 对网络和光模块：很强

HPE 的 Networking 收入 +148.2%，Data Center Networking +233.3%。NVIDIA 同期 Data Center networking 收入 +199%。这说明 AI 交易正在从“买 GPU”扩展为“买网络”。

推论：

- 高速交换机、以太网/InfiniBand、光模块、DSP、SerDes、硅光、光芯片等环节继续有需求弹性。
- 迈威尔科技的 AI 数据中心互连、定制芯片、SerDes/光互连逻辑仍然成立。
- 如果后续市场从训练集群转向推理和企业私有 AI，网络和存储 attach rate 可能比 GPU 单体更重要。

### 4.4 对整机厂：收入强，利润率仍要盯

Dell AI 服务器收入暴涨，但 AI server 本身不一定是最高毛利业务；真正改善利润的关键是：

- 供应链定价能力。
- 存储、服务、软件、融资等 attach rate。
- 规模交付和客户锁定。
- 传统服务器和商用 PC 的同步恢复。

因此，Dell/HPE 后续能否继续涨，不只看 AI 服务器收入，还要看 AI 订单是否转化为更高 EPS 和现金流。

## 5. 后续可能怎么走

### Dell

短期：股价已经对 FY2027 指引上修做了猛烈反应，波动会很大。若后续没有新增订单/供应缓解/指引继续上修，可能出现估值消化或回撤。

中期：只要 AI backlog 仍高、客户数继续扩张、Q2 指引兑现，股价回调后仍可能被资金重新买入。核心验证点是 FY2027 Q2 是否继续保持约 445 亿美元收入、非 GAAP EPS 约 4.80 美元，以及 AI 服务器收入是否继续向全年 600 亿美元目标推进。

风险：memory 供应、组件成本上涨、客户延迟部署、AI server 毛利率偏低、股价一次性涨幅过大。

### HPE

短期：HPE 暴涨主要来自低估值公司突然给出高 EPS/FCF 指引。市场会重新给它估值，但也会质疑 Juniper 并表和协同收益是否可持续。

中期：若 Q3 收入 115-121 亿美元、非 GAAP EPS 0.88-0.93 美元兑现，且 Networking 和 Cloud & AI 利润率继续维持，HPE 的重估逻辑仍在。

风险：Juniper 整合、Networking 高增长含并购因素、Cloud & AI 服务器利润率、企业 AI 订单节奏。

## 6. 这对我们跟踪沪电/生益/迈威尔/美光的意义

Dell/HPE 是下游验证器。它们越强，越说明 AI 基础设施需求正在真实落地，而不是只停留在 NVIDIA 财报里。

映射关系：

| 下游信号 | 最直接受益环节 | 对应公司/方向 |
|---|---|---|
| AI server revenue/backlog 上升 | GPU、HBM、DRAM、NAND、CPU | 美光、存储链 |
| 传统服务器刷新也强 | CPU、DRAM、SSD、硬盘 | 美光、服务器供应链 |
| Data Center Networking 强 | 交换机、光模块、DSP、SerDes | 迈威尔、光互连链 |
| 机柜密度提升 | 高速 PCB、背板、覆铜板、散热、电源 | 沪电股份、生益科技 |
| 企业/主权 AI 客户扩大 | 私有 AI、边缘/推理、存储和服务 | HPE/Dell、存储网络链 |

最重要的结论：AI 产业链的后续发展，可能从“GPU 短缺”扩展到“整套 AI factory 供应链短缺”。如果下游整机厂继续报出订单和 backlog 上修，上游的高端存储、PCB/覆铜板、网络芯片/光互连还有继续验证的空间。

## 7. 后续跟踪清单

- Dell FY2027 Q2：收入是否达到 440-450 亿美元，AI server 全年 600 亿美元目标是否维持或上修。
- Dell：AI backlog 是否继续增长，还是因供给转化而下降。
- HPE FY2026 Q3：收入 115-121 亿美元、非 GAAP EPS 0.88-0.93 美元是否兑现。
- HPE：Networking 高增长中并购因素和内生增长的拆分。
- 美光：下一季指引是否继续反映 memory 供应紧张。
- 沪电/生益：Q2 单季利润是否守住 Q1 平台，调研纪要是否继续提到产能释放和客户认证。
- 迈威尔：数据中心、AI 定制芯片、互连和光电协同是否继续上修。
